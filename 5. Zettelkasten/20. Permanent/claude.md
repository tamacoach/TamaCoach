# Zettelkasten / 20. Permanent Notes

## 목적과 역할
**자신의 고유한 생각과 통찰을 정리**하는 폴더입니다. Permanent Notes는 자신의 핵심 지식 자산으로, 시간이 지나도 가치 있는 아이디어들입니다.

## 어떤 노트가 들어가는가
- 자신의 고유한 아이디어와 통찰
- 여러 자료에서 배운 개념의 종합
- 경험을 통해 얻은 교훈
- 이론과 실제를 연결한 내용
- 자신의 신념과 가치관
- 지속적으로 참고할 개념과 정의

## 작업 방식과 규칙
- **한 노트 = 하나의 아이디어**: 한 가지 개념만 다루기
- **명확하고 자기 언어로 작성**: 다른 사람이 이해할 수 있게
- **타이틀은 명확한 문장**: "교육의 중요성" (X) → "교육은 개인의 관점을 넓힐 수 있다" (O)
- **출처 명시**: 이 아이디어의 시작점이 어디인지 기록
- **상호 연결**: 다른 Permanent Notes와의 연결 고리 만들기
- **파일명**: `[YYYYMMDD] [ID] [명확한 제목]` (예: `20251118 1 교육과 개인성장`)
- **정기적 검토**: 새로운 아이디어로 수정 및 보완
- **이 폴더가 당신의 지식 자산**: 특별히 관리
