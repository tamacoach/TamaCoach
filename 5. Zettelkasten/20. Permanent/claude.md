# 20. Permanent 폴더

## 목적과 역할
자신만의 언어로 작성된 영구적인 지식 노트입니다. Zettelkasten의 핵심이며, 시간이 지나도 가치가 유지되는 아이디어를 담습니다.

## 어떤 노트가 들어가는가
- 완전히 소화된 아이디어
- 자신의 언어로 재해석한 개념
- 여러 원천에서 종합한 인사이트
- 독창적인 생각과 연결

## 작업 방식과 규칙
1. **원자성 원칙**: 하나의 노트에 하나의 아이디어만
2. **자기 표현**: 반드시 자신의 언어로 작성
3. **독립성**: 맥락 없이도 이해 가능하게 작성
4. **연결 필수**: 최소 하나 이상의 다른 노트와 링크
5. **필수 구성요소**:
   - 명확한 제목 (아이디어를 한 문장으로)
   - 본문 설명
   - 관련 노트 링크
   - 출처 (Literature 노트 참조)

## 명명 규칙
- 아이디어를 명확히 표현하는 제목
- 예: `창의성은 연결에서 나온다.md`
- 예: `작은 습관이 큰 변화를 만든다.md`

## 연결 질문
- 이것은 무엇과 관련이 있는가?
- 이것이 반박하는 것은 무엇인가?
- 이것이 지지하는 것은 무엇인가?
- 이것은 어디에 적용될 수 있는가?

## 예시
```
20. Permanent/
├── 지식은 연결될 때 가치가 커진다.md
├── 시스템이 목표보다 중요하다.md
├── 글쓰기는 사고의 도구이다.md
└── 제약이 창의성을 높인다.md
```
