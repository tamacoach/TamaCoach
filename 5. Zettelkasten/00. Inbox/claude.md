# Zettelkasten / 00. Inbox

## 목적과 역할
**빠른 메모와 생각을 임시 저장**하는 폴더입니다. Inbox는 처리되지 않은 정보의 임시 저장소로, 정기적으로 처리되어야 합니다.

## 어떤 노트가 들어가는가
- 떠오른 생각과 아이디어
- 읽다가 멈춘 내용 스크랩
- 빠른 메모와 스니펫
- 나중에 정리할 메모들
- 질문과 고민

## 작업 방식과 규칙
- 제한 없이 빠르게 작성 (나중에 정리하기 위함)
- 형식이나 완성도 신경 쓰지 않음
- **주 1회 이상 Inbox Processing**: 매주 또는 매일 정리 시간 갖기
- 각 항목을 처리:
  - Literature Notes로 변환 (자료 기반)
  - Permanent Notes로 변환 (핵심 아이디어)
  - Projects/Areas로 이동 (실행 항목)
  - 삭제 (불필요한 항목)
- 처리 후 메모는 다른 폴더로 이동
- Inbox는 항상 최소화 상태 유지
