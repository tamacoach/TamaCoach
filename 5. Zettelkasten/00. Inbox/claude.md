# 00. Inbox 폴더

## 목적과 역할
모든 새로운 아이디어, 생각, 메모를 빠르게 캡처하는 임시 저장소입니다. 생각의 진입점입니다.

## 어떤 노트가 들어가는가
- 갑자기 떠오른 아이디어
- 읽은 것에 대한 즉각적인 반응
- 할 일, 리마인더
- 대화에서 얻은 인사이트
- 나중에 처리할 링크나 인용

## 작업 방식과 규칙
1. **빠른 캡처**: 완벽하지 않아도 됨, 일단 기록
2. **최소 형식**: 제목만 있어도 OK
3. **임시 보관**: 이곳에 오래 머물면 안 됨
4. **일일 처리**: 매일 Inbox 비우기
   - Literature로 이동 (원천 자료 관련)
   - Permanent로 발전 (충분히 정제된 아이디어)
   - Projects/Areas로 이동 (행동 관련)
   - 삭제 (불필요한 것)

## 처리 질문
- 이것이 무엇인가?
- 왜 기록했는가?
- 다음 행동은 무엇인가?
- 어디로 이동해야 하는가?

## 목표
**Inbox Zero**: 매일 끝날 때 비어있는 상태 유지

## 예시
```
00. Inbox/
├── 블로그 아이디어 - AI 윤리.md
├── 회의에서 들은 흥미로운 점.md
└── 책 추천 - 친구에게 들음.md
```
