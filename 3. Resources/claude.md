# Resources 폴더

## 목적과 역할
**참고할 수 있는 참고 자료와 정보**를 보관하는 폴더입니다. 직접 생성한 자료가 아닌, 외부에서 수집한 참고 자료들을 저장합니다.

## 어떤 노트가 들어가는가
- 책 요약 및 리뷰
- 웹사이트 및 블로그 스크랩
- 동영상 노트
- 팟캐스트 정리
- 강의 노트
- 유용한 도구 및 라이브러리 정보
- 템플릿과 예시

## 작업 방식과 규칙
- 주제별 또는 분야별 하위 폴더 생성 (예: `Resources/Technology`, `Resources/Design`)
- 출처 명시 필수 (저자, URL, 발행일)
- 자신만의 요약 또는 해석 추가
- 정기적으로 검토하여 불필요한 자료 삭제
- 자주 참고되는 자료는 Permanent Notes로 변환
