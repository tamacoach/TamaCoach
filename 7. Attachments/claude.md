# Attachments 폴더

## 목적과 역할
**이미지, PDF, 동영상 등의 미디어 파일**을 저장하는 폴더입니다. 노트에 포함된 첨부 파일들의 중앙 저장소입니다.

## 어떤 노트가 들어가는가
- 스크린샷 이미지
- 다이어그램 및 차트
- PDF 문서
- 오디오 파일
- 동영상 파일
- 다운로드한 자료

## 작업 방식과 규칙
- 파일명에 날짜와 설명 포함: `[YYYYMMDD] [설명]`
- 파일 유형별 하위 폴더 생성 가능 (예: `Attachments/Images`, `Attachments/PDFs`)
- 노트에서 첨부 파일 참조 시 상대 경로 사용
- 더 이상 사용하지 않는 파일은 정기적으로 삭제
- 큰 파일은 필요시에만 보관 (저장 공간 관리)
- 중요한 PDF나 이미지는 설명과 함께 정리
