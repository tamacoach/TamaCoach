# Attachments 폴더

## 목적과 역할
노트에 첨부되는 모든 파일(이미지, PDF, 오디오 등)을 중앙에서 관리합니다. 볼트를 깔끔하게 유지합니다.

## 어떤 파일이 들어가는가
- 이미지 (PNG, JPG, GIF, SVG)
- PDF 문서
- 오디오 파일
- 비디오 클립
- 기타 첨부 파일

## 작업 방식과 규칙
1. **Obsidian 설정**:
   - 설정 → 파일 및 링크
   - "새 첨부 파일을 만들 위치" → 이 폴더 지정
2. **자동 저장**: 드래그앤드롭 시 자동으로 이 폴더에 저장
3. **명명 규칙**:
   - 설명적인 파일명 사용
   - `YYYY-MM-DD_설명.확장자`
   - 예: `2024-03-15_프로젝트-다이어그램.png`
4. **정리**:
   - 분기별로 사용하지 않는 첨부 파일 정리
   - 링크가 끊어진 파일 삭제

## 하위 폴더 (선택사항)
대량의 첨부 파일이 있다면:
```
Attachments/
├── Images/
├── PDFs/
└── Audio/
```

## 주의사항
- 대용량 파일은 클라우드 동기화 속도에 영향
- 중요한 원본은 별도 백업 권장
- 저작권 있는 자료 관리 주의

## 예시
```
Attachments/
├── 2024-03-15_시스템-구조도.png
├── 2024-03-20_참고-논문.pdf
└── 2024-03-22_인터뷰-녹음.mp3
```
