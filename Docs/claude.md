# Docs 폴더

## 목적과 역할
Claude Code로 Obsidian을 사용할 때 필요한 **참고 문서와 가이드**를 보관하는 폴더입니다. 시스템 사용법과 규칙 문서를 중앙에서 관리합니다.

## 어떤 노트가 들어가는가
- Obsidian 설정 및 사용 가이드
- 폴더 구조 및 네이밍 규칙
- 노트 작성 규칙 및 포맷 가이드
- Claude Code 통합 문서
- 플러그인 및 확장 기능 설명서
- 시스템 유지 보수 체크리스트

## 작업 방식과 규칙
- 이 폴더는 **참고용**이므로 자주 수정하지 않음
- 시스템 변경 사항이 있을 때만 업데이트
- 각 문서는 명확한 제목과 목차 포함
- 스크린샷이나 예시 포함하여 이해하기 쉽게 작성
