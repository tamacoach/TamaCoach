# Templates 폴더

## 목적과 역할
**자주 사용하는 노트 형식**을 템플릿으로 저장하는 폴더입니다. 일관된 형식으로 빠르게 새 노트를 작성하기 위해 사용합니다.

## 어떤 노트가 들어가는가
- 프로젝트 템플릿 (Project Brief, Project Review)
- 독서 템플릿 (Book Notes, Article Summary)
- 일일 노트 템플릿 (Daily Log, Daily Reflection)
- 회고 템플릿 (Weekly Review, Monthly Review)
- 학습 템플릿 (Concept Note, Learning Summary)
- 회의 템플릿 (Meeting Notes, Decision Log)

## 작업 방식과 규칙
- 파일명 접두사: `Template - [이름]`
- 하위 폴더로 분류 가능 (예: `Templates/Project`, `Templates/Learning`)
- Obsidian 템플릿 기능 활용 (Ctrl+Alt+N로 빠르게 삽입)
- 정기적으로 검토하여 실제 사용 중인 템플릿만 유지
- 새로운 템플릿이 필요하면 실제 노트에서 좋은 형식이 나올 때 추출
- 템플릿은 가이드일 뿐, 유연하게 수정하여 사용
